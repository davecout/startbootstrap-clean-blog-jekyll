Une architecture applicative décrit le comportement des applications utilisées dans une organisation, axée sur la façon dont elles interagissent entre elles et avec les utilisateurs. Elle se concentre sur les informations consommées et produites par les applications plutôt que sur leur structure interne comme décrite par une architecture logicielle.
