Une application, ou application logicielle, est un système logiciel utilisé dans les organisations pour faciliter le travail et augmenter la productivité des employés.

Une application utilise les services d'une plateforme applicative pour son exécution. Une application distribuée peu utiliser plusieurs plateformes dont une plateforme applicative, une plateforme d'intégration et une plateforme de données.
